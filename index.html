Git and Github

What is Git?

Git is an open source version control system that allows developers track changes made to their code over time

Git tracks our code by creating snapshots of the code and can be revisited whenever neccessary.

Configuring Git.

git config --global user.name "Praise" is used to configure our name on Git

git config --global user.email "praisebasey442@gmail.com" is used to configure our our email on Gits

Basic Commands

git init is used to initialize Git in your local respository.

git status is used to check the status of the activities of Git in your local respository.

git add "filename" is used to track a single file in git.

git add . is used to track multiple files in git.

git rm --cached "filename"is used to untrack a file in git.

git commit -m "the commit message"is used when we want to save a snapshot of our code. It is like a caption or describtion of the changes made to our code

git remote add origin "repo URL is used to connect your local repo to Github.

git push -u origin master is used to push our code from the local repo to the remote repository.



What is Github

Github is a web based git respository hosting services.

It is a tool that enables collaboration by hosting and sharing gits repositories that team of developers can all contribute to.

While Github uses git, the functionality it provides is entirely different from Gits.

Git is a Version Control System while Github is an online platform for hosting and sharing code.

Why use Github?

  1. Github provides a great way to store your code in a safe and remote location.
  2. Many large open source projects are hosted on Github.
  3. Github encourages collaboration between a team of developers.
